Ship detection in Satellite Images

Description: The motivation behind this individual project was to work on build and analyse the model that performs better to solve this real-life problem. Four models were built. The comparative analysis of all models was done.

Tools and Techniques: Python, Tensorflow

Results: YOLOv3 outperformed both in terms of MAP and Speed of detection
